Meus Projetos

E-commerce Empório Verde

Desenvolvi o Empório Verde, uma plataforma de e-commerce focada em hortifrúti. Utilizando o Bootstrap 5, entrega uma experiência responsiva e intuitiva, permitindo que os clientes naveguem e comprem produtos frescos com facilidade. Destaques do projeto incluem design adaptável, funcionalidade de busca avançada e carrinho de compras integrado. O objetivo principal foi combinar produtos de hortifrúti de alta qualidade com uma experiência de compra online conveniente e segura.

Cadastro de usuários

O componente "Home" é responsável por exibir um formulário de cadastro onde o usuário pode inserir seu nome e idade e, ao clicar no botão "Cadastrar", os dados são enviados para a API de backend e o novo usuário é adicionado à lista de usuários localmente.O componente "Users" é responsável por exibir uma lista de todos os usuários cadastrados. Ele faz uma solicitação à API para obter os dados dos usuários existentes assim que o componente é montado.

Conversor

O Conversor de Moedas é uma aplicação que permite converter valores entre diferentes moedas, incluindo Real Brasileiro (BRL), Dólar Americano (USD), Euro (EUR) e Bitcoin (BTC). Com taxas de câmbio atualizadas através do consumo de API o conversor fornece resultados precisos e em tempo real..

Cronômetro

O cronômetro exibe o tempo decorrido em horas, minutos e segundos, e possui funcionalidades básicas como iniciar, pausar e resetar.O projeto utiliza a função setInterval() do JavaScript para atualizar o cronômetro a cada segundo. Também inclui uma função para formatar os números e sempre exibir dois dígitos, garantindo uma representação visual consistente.

Sorteador

Este é um projeto de sorteio de números aleatórios desenvolvido em JavaScript, HTML e CSS.O usuário pode inserir o valor mínimo e máximo para o sorteio,a página usa JavaScript para gerar um número aleatório dentro do intervalo especificado,os valores mínimo e máximo são arredondados para garantir resultados precisos e o número sorteado é exibido em tempo real na página usando o DOM do JavaScript.

Jokenpô

Neste projeto, criei um jogo de Pedra, Papel e Tesoura, onde os jogadores podem desafiar a máquina,você verá três opções: pedra, papel e tesoura. Clique em uma das opções para fazer sua jogada. A máquina fará sua jogada automaticamente. O resultado do jogo será exibido na tela, juntamente com uma imagem representando o resultado. Os placares de pontos do jogador e da máquina também serão atualizados de acordo com o resultado.
